# hexo-docx

🚀

基于docx-preview的hexo插件

A hexo plugin based on docx-preview which Help you display docx files in hexo

### Install

```shell
npm install hexo-docx
```

**本package的转译预览基于docx-preview**

### Usage/用法:

------

```js
{% docx [src] %}
example:
{% docx https://f2v2.com/example.docx %}
```
